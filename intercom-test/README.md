# Intercom Test

##Problem : Flatten an Array
- - -

### How to run the code

Language used : **Ruby**

Prerequisite:

1. Json Gem   

    ```bash
    $ gem install json
    ```

2. Rspec Gem

    ```bash
    $ gem install rspec
    ```
  
Execute:
    
1. Run the test

    ```bash
    $ rspec spec
    ```
    
2. Run the rake task

    ```bash
    $ rake execute_flatten    
    ```
    
3. Follow instructions on the console
    
You can check the Ruby code here => [ArrayHelper.rb](http://taher.me/intercom_test/)
